TROUBLESHOOTING:
================================================================================

Issue: "Module not found"
Fix: pip install -r requirements.txt

Issue: "Port already in use"
Fix: python app_demo.py --port 5001
  OR: Change port in app_demo.py last line

Issue: Old database errors
Fix: Delete demo_helpdesk.db and restart

Issue: Datetime errors (Windows)
Fix: Already fixed in this professional edition!

PRODUCTION DEPLOYMENT (MySQL):
================================================================================

1. Install MySQL
2. Create database: mysql -u root -p < database/schema.sql
3. Update credentials in app.py (line 25)
4. Run: python app.py
5. Access: http://localhost:5000
